# Qash 💰

**A Smart M-Pesa money manager for Kenya**

Automatically track, categorize, and analyze your M-Pesa transactions with real-time notifications.

---

## ✨ Features

### 🔔 Real-Time Tracking
- **Instant Notifications** - Get notified immediately after any M-Pesa transaction
- **Quick Categorization** - Tap notification to categorize on-the-go
- **Automatic Detection** - Reads M-Pesa SMS and extracts transaction details

### 🧠 Smart Categorization
- **Auto-Learn** - Remembers your preferences for each merchant
- **60+ Pre-loaded Merchants** - KPLC, Naivas, Uber, KFC, Safaricom, and more
- **11 Categories** - Transport, Food, Bills, Entertainment, Airtime, Shopping, Health, Dining, Government, Personal Transfer, Other

### 💸 Fuliza Support
- **Separate Tracking** - Fuliza fees tracked independently
- **Negative Balance** - Shows your Fuliza debt clearly
- **Smart Parsing** - Handles dual Fuliza SMS correctly

### 📊 Analytics Dashboard
- **Current Balance** - Real M-Pesa balance from latest transaction
- **Today's Spending** - Track daily expenses
- **Weekly Overview** - See spending trends
- **Transaction History** - Complete searchable list

### 📥 Bulk Import
- **Import All** - Load all past M-Pesa messages at once
- **Duplicate Prevention** - Never import the same transaction twice
- **Progress Tracking** - See import status in real-time

---

## 🎯 How It Works

### First Time Setup
1. Open Qash and grant SMS permissions
2. Tap "Import M-Pesa" to load past transactions
3. Categorize any unknown merchants
4. Done! Future transactions auto-categorize

### Daily Use
1. Make any M-Pesa transaction (send money, buy airtime, etc.)
2. Qash detects the SMS and shows a notification
3. Tap notification to confirm/change category
4. Transaction automatically saved to your history

### Supported Transactions
✅ Send Money | ✅ Receive Money | ✅ PayBill | ✅ Buy Goods (Till)  
✅ Lipa na M-Pesa | ✅ Airtime | ✅ Withdraw | ✅ Fuliza
